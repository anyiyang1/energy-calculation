这个网页是一个纯前端应用（Single Page Application），所有的数据都存储在你的浏览器本地。并没有发送到任何远程服务器或数据库。

数据的查看方式分为以下两种情况：

1. 查看你自己保存的“项目数据”
当你点击“新建项目”或“保存计算”时，数据被存储在浏览器的 Local Storage（本地存储） 中。

如何查看原始数据：

在网页上按键盘上的 F12 键（或右键点击页面 -> “检查/Inspect”）。

在打开的开发者工具面板上方，点击 Application（应用）标签页（如果没有看到，可能折叠在 >> 里面）。

在左侧菜单栏找到 Local Storage，展开并点击下方的链接（通常是 file:// 或 http://...）。

在右侧列表中，找到 Key 为 energy_projects 的一项。

这就是你所有项目的原始 JSON 数据。

⚠️ 重要提示：

数据都在本地：如果你更换电脑、更换浏览器，或者使用了“无痕浏览模式”，是看不到之前保存的数据的。

清理缓存需谨慎：如果你清理了浏览器的“Cookies 和其他网站数据”，这些项目数据会被永久删除。

备份建议：建议利用网页内的“导出 Excel”功能，将重要计算结果保存为文件备份。
